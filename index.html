Практические навыки

Brackets

* Использовала Brackets для создания простых html документов с присоединением css.

* Создавала отчет по качеству в конце тестирования.

Jira

* Использовала для баг-трэкинга.

* Применяла язык Jquery для фильтрации и поиска нужных репортов.

* Можно отследить состояние тикета (открыт\закрыт\в работе).

Fiddler

* Использовала для перехвата и изменения запросов от\к серверу.

MySQL

* Создавала простые запросы, используя селекторы JOIN, INSERT INTO, COUNT, MAX, MIN.
